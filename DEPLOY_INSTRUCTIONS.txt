STEP 1: CREATE GITHUB REPOSITORY
─────────────────────────────────────────────────────────────────────

1. Open your browser and go to: https://github.com/new

2. Fill in the form:
   Repository name: Streamlit_Duplicate_Detector
   Description: AI-powered PDF panel duplicate detection with modality routing
   Visibility: Public (or Private - either works)
   
   ⚠️ IMPORTANT: Do NOT check these boxes:
   - [ ] Add a README file
   - [ ] Add .gitignore
   - [ ] Choose a license

3. Click "Create repository"

4. Copy the repository URL from the next page
   It will look like: https://github.com/YOUR_USERNAME/Streamlit_Duplicate_Detector.git


STEP 2: PUSH TO GITHUB
─────────────────────────────────────────────────────────────────────

Once you have your repository URL, run these commands:

cd "/Users/zijiefeng/Desktop/Guo's lab/APP/Streamlit_Duplicate_Detector"

# Add your GitHub repo as remote (REPLACE with your actual URL)
git remote add origin https://github.com/YOUR_USERNAME/Streamlit_Duplicate_Detector.git

# Push your code
git branch -M main
git push -u origin main

✅ After this, your code will be on GitHub!


STEP 3: DEPLOY TO STREAMLIT CLOUD
─────────────────────────────────────────────────────────────────────

1. Go to: https://share.streamlit.io/

2. Sign in with your GitHub account

3. Click "New app" button

4. Fill in the deployment form:
   
   Repository: YOUR_USERNAME/Streamlit_Duplicate_Detector
   Branch: main
   Main file path: streamlit_app.py
   
   App URL: (choose a custom subdomain or use default)

5. Click "Deploy!"

6. Wait ~5-10 minutes for deployment to complete

✅ Your app will be live at: https://YOUR_APP_NAME.streamlit.app/


STEP 4 (OPTIONAL): UPDATE REQUIREMENTS IF NEEDED
─────────────────────────────────────────────────────────────────────

If deployment fails due to dependencies, you can update requirements.txt
by adding package versions:

Current requirements.txt should have:
- streamlit
- pandas
- numpy
- opencv-python-headless
- Pillow
- scikit-image
- torch
- torchvision
- open-clip-torch
- tqdm
- pymupdf
- python-magic-bin (Windows only, remove for Linux/Mac deployment)


TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────

If Streamlit Cloud shows "Python version not supported":
- Add a file named "runtime.txt" with content: python-3.12

If deployment is slow or times out:
- Check Streamlit Cloud logs for specific errors
- Verify all dependencies in requirements.txt are spelled correctly

If "module not found" errors:
- Add the missing package to requirements.txt
- Streamlit Cloud will automatically reinstall
